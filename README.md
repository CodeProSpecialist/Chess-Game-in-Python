# Chess-Game-in-Python
Chess game in Python

This was tested to work on Debian GNU/Linux 12 (bookworm). 

To install:

chmod +x install.sh

./install.sh

To play a chess game, open a comand line terminal and type the following command:

python3 play-chess.py


---------------------------------------------------------------------------
To uninstall:

chmod +x uninstall.sh

./uninstall.sh
